# Journal des versions

Ce fichier résume les changements importants du manuel. Les petites corrections de ponctuation ne sont pas forcément listées.

## 0.1.0 - 12 juillet 2026

Première base publique du projet.

### Ajouté

- une page d'accueil autonome et lisible sur mobile ;
- une méthode d'apprentissage basée sur l'observation et le debug ;
- les fondations de GameMaker et du GML ;
- le déplacement top-down et la normalisation des diagonales ;
- les animations directionnelles et la variable `facing` ;
- les fonctions et leurs paramètres ;
- la caméra pixel art et la différence entre position logique et visuelle ;
- une introduction aux collisions avec objets et tilemaps ;
- le Draw GUI, les PV, les dégâts et le Game Over ;
- une introduction à Git pour les projets GameMaker ;
- les premiers liens vers le manuel officiel.
